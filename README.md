## xiao_note

- numpy_random: numpy随机抽样

- pandas_vs_dplyr: pandas与dplyr方法比较

- lr_q-learning: 增强学习: Q-Learning
- ml_leanring_validation_curve: 学习曲线与验证曲线

- num_ops_image: 基于卷积模型的数字与操作符图像识别模型
- transfer_learning: 迁移学习 —— 基于xception模型的图像分类
